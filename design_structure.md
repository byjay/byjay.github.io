# 오토캐드 강좌 블로그 홈페이지 구조 설계

## 1. 사이트 구조 (Site Map)

```
홈페이지 (Home)
├── 기초 강좌 (Basic Tutorials)
│   ├── 1. 오토캐드 소개 및 인터페이스
│   ├── 2. 기본 도형 그리기
│   ├── 3. 객체 선택 및 수정
│   ├── 4. 레이어 관리
│   └── 5. 치수 및 주석
├── 심화 강좌 (Advanced Tutorials)
│   ├── 블록 및 속성
│   ├── 3D 모델링 기초
│   ├── 플롯 및 출력
│   └── 사용자 정의 설정
├── 팁 & 노하우 (Tips & Tricks)
│   ├── 단축키 모음
│   ├── 생산성 향상 팁
│   └── 오류 해결 가이드
├── 자료실 (Resources)
│   ├── 예제 파일 다운로드
│   ├── 템플릿 파일
│   └── 유용한 링크
└── 소개 (About)
    ├── 블로그 소개
    └── 연락처
```

## 2. 페이지별 기능 및 콘텐츠

### 2.1 홈페이지 (Home)
- **헤더**: 로고, 네비게이션 메뉴, 검색 기능
- **히어로 섹션**: 블로그 소개, 주요 강좌 링크
- **최신 강좌**: 최근 업데이트된 강좌 목록 (카드 형태)
- **인기 강좌**: 조회수가 높은 강좌 목록
- **팁 섹션**: 오늘의 팁 또는 주요 팁 하이라이트
- **푸터**: 저작권, 소셜 미디어 링크, 사이트맵

### 2.2 강좌 페이지 (Tutorial Pages)
- **브레드크럼**: 현재 위치 표시
- **강좌 제목 및 메타 정보**: 작성일, 난이도, 예상 소요 시간
- **목차**: 강좌 내용의 섹션별 링크
- **본문 콘텐츠**: 텍스트, 이미지, 코드 블록
- **네비게이션**: 이전/다음 강좌 링크
- **관련 강좌**: 유사한 주제의 강좌 추천

### 2.3 카테고리 페이지 (Category Pages)
- **카테고리 소개**: 해당 카테고리에 대한 설명
- **강좌 목록**: 그리드 또는 리스트 형태로 강좌 표시
- **필터링**: 난이도, 작성일 등으로 정렬
- **페이지네이션**: 강좌가 많을 경우 페이지 분할

## 3. 디자인 컨셉

### 3.1 컬러 팔레트
- **주 색상**: #2563EB (파란색) - 신뢰성과 전문성 표현
- **보조 색상**: #64748B (회색) - 텍스트 및 보조 요소
- **강조 색상**: #F59E0B (주황색) - 버튼, 링크 등 액션 요소
- **배경 색상**: #F8FAFC (연한 회색) - 깔끔한 배경
- **텍스트 색상**: #1E293B (진한 회색) - 가독성 향상

### 3.2 타이포그래피
- **제목**: Noto Sans KR Bold (한글), Inter Bold (영문)
- **본문**: Noto Sans KR Regular (한글), Inter Regular (영문)
- **코드**: JetBrains Mono (고정폭 폰트)

### 3.3 레이아웃 원칙
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 최적화
- **그리드 시스템**: 12컬럼 그리드 기반 레이아웃
- **여백**: 충분한 여백으로 가독성 향상
- **일관성**: 모든 페이지에서 일관된 디자인 패턴 적용

## 4. 사용자 경험 (UX) 고려사항

### 4.1 네비게이션
- **직관적인 메뉴 구조**: 사용자가 쉽게 원하는 콘텐츠를 찾을 수 있도록
- **검색 기능**: 키워드로 강좌 및 팁 검색 가능
- **브레드크럼**: 현재 위치를 명확히 표시

### 4.2 콘텐츠 접근성
- **목차**: 긴 강좌의 경우 섹션별 바로가기 제공
- **진행률 표시**: 강좌 읽기 진행률 시각화
- **관련 콘텐츠**: 현재 강좌와 관련된 다른 강좌 추천

### 4.3 성능 최적화
- **이미지 최적화**: WebP 형식 사용, 지연 로딩 적용
- **코드 분할**: 필요한 JavaScript만 로드
- **캐싱**: 정적 자원에 대한 브라우저 캐싱 활용

## 5. 기술적 구현 방향

### 5.1 프론트엔드
- **HTML5**: 시맨틱 마크업으로 접근성 향상
- **CSS3**: Flexbox, Grid를 활용한 반응형 레이아웃
- **JavaScript**: 인터랙티브 요소 및 검색 기능 구현
- **Progressive Web App**: 오프라인 접근 가능성 고려

### 5.2 콘텐츠 관리
- **Markdown**: 강좌 콘텐츠를 Markdown 형식으로 작성
- **JSON**: 강좌 메타데이터 및 카테고리 정보 관리
- **정적 사이트**: 빠른 로딩과 SEO 최적화

### 5.3 반응형 브레이크포인트
- **모바일**: 320px ~ 767px
- **태블릿**: 768px ~ 1023px
- **데스크톱**: 1024px 이상

이러한 구조와 디자인 원칙을 바탕으로 사용자 친화적이고 전문적인 오토캐드 강좌 블로그를 구축할 예정입니다.
